JS Project 2

In this project we were tasked with choosing a API and pulling information from it to do something. I decided to pull information from the school directory 2020. I used that information to allow the user to input a city and search what schools have certain things in that city. For example if the user put Brooklyn, they can then choose what club they want to go to or what language program they want to take. Then after they put in that information it will provide them with shcools that meet the criteria.

I ran in with a lot of troubles in the project especially when pulling information. The information in the API weren't seperate, for example if I wanted to pull the sports a school provides it would be "Basketball, Baseball, Volleyball" etc but I wanted "Basketball", "Basebll", "Volleyball" so I can pull each sport and not get the whole thing. I used the function split and push to help me with that but there was another problem. I had to loop though a array within a array so I solved that by using 2 loops.

Next time I do a project I want to try incorporating more visuals in order to make the project more appealing. I also want to incorporate more CSS and HTML into my JS.
